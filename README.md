# 🌾 Portal Agrícola Tech - Agrinho 2026

## 📌 Tema
**Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente**

## 🎯 Objetivo
Calculadora interativa de produtos agrícolas sustentáveis que permite:
- Selecionar produtos orgânicos
- Ajustar preços por unidade
- Calcular quantidades
- Gerenciar carrinho de compras
- Visualizar impacto ambiental (CO₂)

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (p5.js)

## 🚀 Funcionalidades
- ✅ Teclado numérico virtual
- ✅ Seleção de 3 produtos sustentáveis
- ✅ Ajuste de preço personalizado
- ✅ Carrinho de compras completo
- ✅ Remoção de itens individuais
- ✅ Cálculo automático do valor total
- ✅ Indicador de impacto ambiental
- ✅ Interface responsiva
- ✅ Design com tema natureza

## 🌱 Impacto Ambiental
Cada produto possui classificação de impacto de CO₂:
- 🌱 Sementes Orgânicas: Baixo impacto
- 🌿 Adubo Natural: Impacto muito baixo
- 🌳 Mudas Nativas: Impacto negativo (captura CO₂)
